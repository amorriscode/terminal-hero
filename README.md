## Getting Started

1. Create and activate a `virtualenv` with python:

```bash
python3 -m virtualenv th-env
source ./th-env/bin/activate
```

2. Install dependencies

```bash
pip install -r requirements.txt
```

3. Rock out!

```bash
python main.py [songPath]
```
